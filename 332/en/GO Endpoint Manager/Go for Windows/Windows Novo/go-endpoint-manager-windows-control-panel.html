This feature allows the elevation of privilege for Windows Control Panel applications.

GO Endpoint Manager Core - Control Panel

Execute an application in the control panel

  1. Access the user's desktop.
  2. Start Core.
  3. Click Control Panel.
  4. Select an item from the list.
  5. Right-click on it.
  6. Execute.
Caution

This configuration option is available in the following versions:

✖ Legacy             ✔3.27 and later
